# 하스켈 프로그래밍 배우기

"하스켈 프로그래밍 배우기"는 '**프로그래밍, 수리논리, 수학은 하나다**' 라는 주제하에 진행되는 
강좌 시리즈의 첫번째 강좌입니다. 

학습 내용 및 프로그래밍 환경, 도커(Docker), 주피터 노트북 등 학습환경과 관련된 자세한 사항은 
[강좌 홈페이지](https://prologma.github.io/learn-you-a-haskell/)에서
확인할 수 있습니다.
